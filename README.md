## Projects

### 1. Credit Default Prediction
- **Description:** Built a machine learning model to predict the probability of financial distress using 150,000 real borrower records
- **Model:** Gradient Boosting Classifier
- **Result:** 0.866 AUC score — top tier on Kaggle's Give Me Some Credit competition
- **Key Finding:** Payment history (90+ days late) is 52% more predictive of default than income
- **Kaggle Notebook:** [View Here](https://www.kaggle.com/code/emigdiotormolopez/notebook89682c577a)
